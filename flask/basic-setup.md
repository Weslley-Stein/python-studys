# Flask Basic Setup

### Importing Flask
```py
#On .py File
from flask import Flask
```

### Instancianting the App
```py
#On .py file
app = Flask(__name__)
```

### Exporting venv
```bash
#On bash terminal
export FLASK_APP=file_name 
```

### Running the App
```bash
#On bash terminal
flask run
```

### Able debug mode
```bash
#On bash terminal
export FLASK_DEBUG=1
```

### Disable Debug Mode
```bash
#On bash terminal
export FLASK_DEBUG=0
```
